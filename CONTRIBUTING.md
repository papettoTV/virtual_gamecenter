# Contributing

このリポジトリでは、ゲーム本体・Webサイト・バックエンドの改善提案を歓迎します。

## ライセンスへの同意

Pull Request を送ることで、その貢献物をこのリポジトリの `LICENSE` に記載された MIT License の下で提供することに同意したものとします。

投稿者は、Pull Request に含めるコード・文章・設定・その他の成果物について、投稿する権利を持っている必要があります。

## 素材の扱い

画像、音楽、効果音、フォント、キャラクター、タイトルロゴ、ゲームデザイン由来の素材などは、コードとは別に権利確認が必要です。

Pull Request に素材を含める場合は、以下のいずれかを満たしてください。

- 自作素材である
- 利用許諾を得ている
- ライセンス上、このプロジェクトで利用・改変・再配布できる

第三者のゲーム、アニメ、漫画、映画、音楽、商標、キャラクター等に由来する素材を、許諾なく追加しないでください。

## Pull Request の目安

- 変更の目的を説明してください。
- ゲームバランス変更の場合は、何を狙った調整かを書いてください。
- 新しい素材を追加する場合は、出典・作者・ライセンス・利用条件を書いてください。
- 動作確認した範囲を記載してください。

## 受け入れ判断

Pull Request は、ゲーム体験、保守性、権利面、安全性を確認した上で受け入れます。

権利関係が不明な素材や、ライセンス条件がこのプロジェクトと合わない素材は、受け入れない場合があります。
