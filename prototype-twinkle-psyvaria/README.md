# Graze Duel Prototype

「ティンクルスタースプライツ x サイヴァリア」を仮説にした、弾幕かすり・無敵体当たり・ボス撃破型シューティングの試作です。

## 遊び方

ローカルサーバで起動します。

```bash
npm install
npm run dev
```

スマホで遊ぶ場合は、PCとスマホを同じWi-Fiにつなぎ、起動時に表示される `Network` のURLをスマホのブラウザで開きます。

公開前のビルド確認:

```bash
npm run build
npm run preview
```

Cloudflare Pagesへ反映:

```bash
npm run deploy
```

ローカル開発中はBasic認証を無効化し、Cloudflare Pages上ではBasic認証を有効にしています。

ローカル起動後は、ゲームセンタートップから `Graze Duel` を選び、筐体画面で `ゲームスタート` を押すとソロプレイが始まります。

同じURLを別のブラウザまたはスマホでも開くと、先に筐体へ入った画面がプレイヤー、後から入った画面が観戦者になります。プレイヤーがゲームを開始すると、観戦者側の `観戦する` ボタンからゲーム状態をリアルタイムで確認できます。

- ローカル観戦は約10fpsの状態同期です。
- 観戦画面からゲーム操作はできません。
- プレイヤーが筐体を離れると、観戦画面は筐体画面へ戻ります。

- 移動: 矢印キー / WASD
- スマホ移動: ゲーム画面内をタップし、その位置を中心にスライドした方向へ移動
- スマホ表示: 自分の画面を大きく表示し、相手は右上の情報HUDで表示
- スマホ操作: 画面下のリスタート/一時停止ボタンを使用
- 低速移動: Shift
- 攻撃: ゲージが溜まると自動発射
- 一時停止/再開: Space
- リスタート: R

## ルール

- 敵弾をギリギリで避けるとスコアとゲージが増えます。
- ゲーム開始直後は、少し間を置いてから弾が出始めます。
- 通常弾は、直進、狙い撃ち、斜め弾、3連弾、扇状弾、弾幕の壁、狙い連射など複数パターンがランダムに混ざります。
- 通常弾パターンは完全に毎回バラバラではなく、一定回数まとまって出てから次のパターンに切り替わります。
- スコアは倍率の影響を受けず、かすり1回ごとに固定値で増えます。
- かすった瞬間に短い効果音が鳴ります。
- 短い間に連続でかするとコンボになり、倍率が上がります。
- 最初はレベル1で、ゲージが満タンになるたびにレベルが1つ上がります。
- レベルが上がるほど、次のレベルアップに必要なゲージ量が増えます。
- レベルアップ直後は2.4秒無敵になり、無敵中のレベルアップでは0.4秒だけ延長されます。
- 無敵リングは残り時間に応じて小さくなります。
- 無敵終了直前の0.5秒は、リングが透明点滅します。
- ゲージが溜まると、効果音とともに自動で相手側に弾幕を送ります。
- レベルが上がるほど、攻撃時に送る弾の量が増えます。
- 攻撃弾のパターンは今後検討するため、現時点では従来の攻撃弾を維持しています。
- ボスはレベル到達ごとに登場します。
- レベル10で丸型のレベル1ボスが登場し、画面中央にじっとしています。
- 次の中ボスは、前のボスを倒した時点のレベル+10で登場します。
- 最後のラスボスも、中ボスを倒した時点のレベル+10で登場します。
- 中ボスは逆三角形で、左右にゆっくり往復します。
- ラスボスは星型で、左右に往復しながら上下にも細かく動きます。
- ボスには弾やかすりではダメージを与えられません。
- 無敵中のリングをボスに当てた時だけ、ボスにダメージを与えます。
- ボスにダメージを与えるたび、残り無敵時間が短くなります。
- ボスを倒すたびに画面内の弾がすべて消え、次のボスが登場します。
- ボス撃破時は、数秒間ゲーム全体がスローになります。
- ラスボスを倒すとクリアです。
- 自機の当たり判定は中心の小さな点だけです。
- 自分のライフは3で、被弾するとライフを1失います。
- 被弾時は軽い爆発演出と効果音が出て、その後数秒間点滅しながら無敵になります。
- レベルが10上がるごとに、通常攻撃に加えて大きなボス弾を送ります。
- 自分が被弾した時点でゲームオーバーです。

## 評価したいこと

- かすり続ける行為が気持ちいいか。
- 無敵中にボスへ突っ込む判断が気持ちいいか。
- ボスに攻撃するほど無敵が短くなる緊張感があるか。
- クリアを目指して「もう一度やりたい」と思えるか。

## デバッグ機能

- 敵弾量スライダーで、基本の敵弾量を1〜10段階で調整できます。
- 現在の通常量は `2 / 10` として設定しています。
- 自機の当たり判定トグルで、被弾あり/なしを切り替えられます。
- ゲージ増加難度の上下ボタンで、レベルごとの必要ゲージ増加量を調整できます。
- HUDに現在ゲージとレベルアップに必要な最大ゲージを表示します。
